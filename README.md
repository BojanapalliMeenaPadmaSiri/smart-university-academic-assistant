# Smart University Academic Assistant

## Overview
The Smart University Academic Assistant is an AI-powered chatbot developed using Python and Google Gemini 2.5 Flash. It helps students obtain information related to admissions, attendance, examinations, fee payments, internships, placements, and faculty support.

## Features
- Admission Assistance
- Attendance Information
- Examination Support
- Fee Payment Guidance
- Internship Information
- Placement Assistance
- AI-Powered Natural Language Responses

## Technologies Used
- Python
- Google Gemini API
- Gemini 2.5 Flash
- Natural Language Processing (NLP)
- Large Language Models (LLMs)
- Visual Studio Code

## Project Architecture

Student Query
↓
Gemini API
↓
Gemini 2.5 Flash LLM
↓
Generated Response

## Installation

1. Clone the repository
2. Install dependencies

```cmd
pip install -r requirements.txt
```

3. Add your Gemini API Key in chatbot.py
4. Run the application

```cmd
python chatbot.py
```

## Sample Queries

- What is the attendance requirement?
- Tell me about placements.
- How can I apply for admission?
- What are the fee payment methods?
